## Catch of the Day

This app has been created over the course of ["React For Beginners"][1] from WesBos.

You can ["try it out here"][2]

Instructions how to launch this project:

1. git clone https://github.com/mrOrlando/CatchOfTheDay
2. cd CatchOfTheDay
3. npm install
4. npm run start
